More than HTML: Web Development Using Modern Tools

Travel site

I signed up for a Udemy course titled "Git a Web Developer Job: Mastering the Modern Workflow" hoping to tighten up my git skills and refresh my web development abilities. Not only did I learn git, but I also learned an entire host of new web development tools that I could use to make my life a web developer more streamlined and organized.

Throughout the course we focused on building a simple website that used modern techniques and tools. See below for a summary of what was covered:

  • Block Element Modifier (BEM) Methodology
  • Mobile First Design
  • Using npm to download useful packages to aid in web development
  • Organized CSS using PostCSS which includes use of variables, mixins, and nesting
  • Using Gulp to build tasks that automate browser refresh, css post processing, and javascript file generation
  • Responsive images tailored to device size
  • Webpack and Babel
  • Waypoints for detecting scroll events and triggering actions
